# ramenjutsu

### Setup
- Make sure to install React and Next.js `npm install --save next react react-dom`
- `npm install`

### _Setup static export(optional)_
- `npm install -g serve`

#### Run local
- `npm run dev`

#### Run build/export(_optional to preview production locally_)
- `npm run build`
- `npm run export`
- `cd out`
- `serve -p 8080`
